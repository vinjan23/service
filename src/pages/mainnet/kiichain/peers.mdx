Automatically adding Live peers to your config.toml file
```
peers="$(curl -sS https://rpc-m.kiichain.vinjan-inc.com:443/net_info | jq -r '.result.peers[] | "\(.node_info.id)@\(.remote_ip):\(.node_info.listen_addr)"' | awk -F ':' '{print $1":"$(NF)}' | sed -z 's|\n|,|g;s|.$||')"
sed -i -e "s/^persistent_peers *=.*/persistent_peers = \"$peers\"/" $HOME/.kiichain/config/config.toml
```
### Seed

seeds="8c5a218a5b63c094e4172d3c674a8627c4f5335e@seed-kiichain.vinjan-inc.com:19956"
