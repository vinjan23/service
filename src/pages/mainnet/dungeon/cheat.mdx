### CLI Cheatsheet

- Always be careful with the capitalized words
- Specify `--chain-id`

### Wallet Management

Add Wallet
Specify the value `<wallet>` with your own wallet name

```
dungeond keys add wallet
```

Recover Wallet
```
dungeond keys add wallet --recover
```

List Wallet
```
dungeond keys list
```

Delete Wallet
```
dungeond keys delete wallet
```

Check Wallet Balance
```
dungeond q bank balances $(dungeond keys show wallet -a)
```

### Validator Management

Please adjust `<wallet>` , `MONIKER` , `YOUR_KEYBASE_ID` , `YOUR_DETAILS` , `YOUUR_WEBSITE_URL`

Create Validator (Staking)
```
dungeond tendermint show-validator
```
```
nano /root/.dungeonchain/validator.json
```
```
{
  "pubkey": ,
  "amount": "1000000000udgn",
  "moniker": "",
  "identity": "",
  "website": "",
  "security": "",
  "details": "",
  "commission-rate": "0.05",
  "commission-max-rate": "0.2",
  "commission-max-change-rate": "0.01",
  "min-self-delegation": "1"
}
```
```
dungeond tx staking create-validator $HOME/.dungeonchain/validator.json \
--from wallet \
--chain-id dungeon-1 \
--gas-prices=0.05udgn \
--gas-adjustment=1.5 \
--gas=auto
```


Edit Validator
```
dungeond tx staking edit-validator \
--new-moniker="" \
--identity="" \
--website="" \
--details="" \
--chain-id=dungeon-1 \
--commission-rate="0.02" \
--from=wallet \
--gas=auto \
--gas-adjustment=1.5 \
--gas-prices=0.05udgn
```

Unjail Validator
```
dungeond tx slashing unjail --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Check Jailed Reason
```
dungeond query slashing signing-info $(dungeond tendermint show-validator)
```

### Token Management

Withdraw Rewards
```
dungeond tx distribution withdraw-all-rewards --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Withdraw Rewards with Comission
```
dungeond tx distribution withdraw-rewards $(dungeond keys show wallet --bech val -a) --commission --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Delegate Token to your own validator
```
dungeond tx staking delegate $(dungeond keys show wallet --bech val -a) 1000000000udgn --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Delegate Token to other validator
```
dungeond tx staking redelegate $(dungeond keys show wallet --bech val -a) <TO_VALOPER_ADDRESS> 1000000000udgn --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Unbond Token from your validator
```
dungeond tx staking unbond $(dungeond keys show wallet --bech val -a) 1000000000udgn --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

Send Token to another wallet
```
dungeond tx bank send wallet <TO_WALLET_ADDRESS> 1000000000udgn --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

### Governance 

Vote
You can change the value of `yes` to `no`,`abstain`,`nowithveto`

```
dungeond tx gov vote 1 yes --from wallet --chain-id dungeon-1 --gas-adjustment=1.5 --gas=auto --gas-prices="0.05udgn"
```

### Other

Set Your own Custom Ports
You can change value `CUSTOM_PORT=44` To any other ports

- PORT=105
```  
PORT=165
sed -i -e "s%:26657%:${PORT}57%" $HOME/.dungeonchain/config/client.toml
sed -i -e "s%:26658%:${PORT}58%; s%:26657%:${PORT}57%; s%:6060%:${PORT}60%; s%:26656%:${PORT}56%; s%:26660%:${PORT}61%" $HOME/.dungeonchain/config/config.toml
sed -i -e "s%:1317%:${PORT}17%; s%:9090%:${PORT}90%" $HOME/.dungeonchain/config/app.toml
```

Enable Indexing usually enabled by default
```
sed -i 's|^indexer *=.*|indexer = "kv"|' $HOME/.dungeonchain/config/config.toml
```

Disable Indexing
```
sed -i 's|^indexer *=.*|indexer = "null"|' $HOME/.dungeonchain/config/config.toml
```

Reset Chain Data
```
dungeond tendermint unsafe-reset-all --home $HOME/.dungeonchain --keep-addr-book
```

### Delete Node

WARNING! Use this command wisely 
Backup your key first it will remove Defund from your system

```
sudo systemctl stop dungeond
sudo systemctl disable dungeond
sudo rm /etc/systemd/system/dungeond.service
sudo systemctl daemon-reload
rm -f $(which dungeond)
rm -rf $HOME/.dungeonchain
rm -rf $HOME/dungeonchain
```
