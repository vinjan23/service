### Hardware requirements

The following requirements are recommended:

- 4 or more physical CPU cores
- At least 200GB of SSD disk storage
- At least 8GB of memory (RAM)
- At least 100mbps network bandwidth


### Install dependencies Required

```
sudo apt update && sudo apt upgrade -y && sudo apt install curl tar wget clang pkg-config libssl-dev jq build-essential bsdmainutils git make ncdu gcc git jq chrony liblz4-tool -y
```

### Install go

We are gonna use GO Version 1.24.5
If you already have GO installed you can skip this 

```
ver="1.24.5"
cd $HOME
wget "https://golang.org/dl/go$ver.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$ver.linux-amd64.tar.gz"
rm "go$ver.linux-amd64.tar.gz"
echo "export PATH=$PATH:/usr/local/go/bin:$HOME/go/bin" >> ~/.bash_profile
source ~/.bash_profile
go version
```

### Install binary

```
cd $HOME
mkdir -p $HOME/.gnodi
wget https://github.com/gnodi-network/gnodi/releases/download/v1.0.1/gnodid-linux
chmod +x gnodid-linux
sudo mv gnodid-linux /usr/local/bin/gnodid
```

### Init `Change <MONIKER> with ur Name`

```
gnodid init <MONIKER> --chain-id gnodi
```

### Download Genesis file and addrbook

- Genesis

```
curl -L https://snapshot.vinjan-inc.com/gnodi/genesis.json > $HOME/.gnodi/config/genesis.json
```
- Addrbook

```
curl -L https://snapshot.vinjan-inc.com/gnodi/addrbook.json > $HOME/.gnodi/config/addrbook.json
```

### Port
```
PORT=155
sed -i -e "s%:26657%:${PORT}57%" $HOME/.gnodi/config/client.toml
sed -i -e "s%:26658%:${PORT}58%; s%:26657%:${PORT}57%; s%:6060%:${PORT}60%; s%:26656%:${PORT}56%; s%:26660%:${PORT}60%" $HOME/.gnodi/config/config.toml
sed -i -e "s%:1317%:${PORT}17%; s%:9090%:${PORT}90%" $HOME/.gnodi/config/app.toml
```

### Set Gas
```
sed -i 's/minimum-gas-prices = ""/minimum-gas-prices = "0.25uGNOD"/' ~/.gnodi/config/app.toml
```

### Config pruning
```
sed -i \
-e 's|^pruning *=.*|pruning = "custom"|' \
-e 's|^pruning-keep-recent *=.*|pruning-keep-recent = "100"|' \
-e 's|^pruning-keep-every *=.*|pruning-keep-every = "0"|' \
-e 's|^pruning-interval *=.*|pruning-interval = "20"|' \
$HOME/.gnodi/config/app.toml
```

### Indexer Off
```
sed -i 's|^indexer *=.*|indexer = "null"|' $HOME/.gnodi/config/config.toml
```

### create service file and start node

```
sudo tee /etc/systemd/system/gnodid.service > /dev/null <<EOF
[Unit]
Description=gnodi
After=network-online.target
[Service]
User=$USER
ExecStart=$(which gnodid) start
Restart=on-failure
RestartSec=3
LimitNOFILE=65535
[Install]
WantedBy=multi-user.target
EOF
```

- Enable Service and Start Node

```
sudo systemctl daemon-reload
sudo systemctl enable gnodid
```
